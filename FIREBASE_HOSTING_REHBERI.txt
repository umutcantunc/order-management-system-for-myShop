FIREBASE HOSTING'E YÜKLEME:
============================

1. Firebase CLI'yi kurun (eğer yoksa):
   npm install -g firebase-tools

2. Firebase'e giriş yapın:
   firebase login

3. Projeyi Firebase'e bağlayın:
   firebase init hosting

4. Build klasörünü seçin:
   - Public directory: build/web
   - Single-page app: Yes
   - Automatic builds: No (manuel build)

5. Web uygulamasını build edin:
   flutter build web --release

6. Firebase Hosting'e deploy edin:
   firebase deploy --only hosting

7. URL'nizi paylaşın:
   https://[PROJECT-ID].web.app
   veya
   https://[PROJECT-ID].firebaseapp.com

iOS KULLANICILARI İÇİN TALİMATLAR:
===================================

1. Safari'de uygulamayı açın
2. Paylaş butonuna (kare ok) tıklayın
3. "Ana Ekrana Ekle" seçeneğini seçin
4. İkon ve ismi kontrol edin
5. "Ekle" butonuna tıklayın
6. Ana ekrandan uygulamayı açın
7. Standalone modda çalışacak (adres çubuğu yok!)
